## To run this project make
```
pip install django
```
### Before running server set an enviroment variable:
  - For Windows:
    ```
    $env:TEST_DATA="your data"
    ```
  - For Linux, MacOS:
    ```
    export TEST_DATA="your data"
    ```
### Run project:
  ```
  python3 manage.py runserver
  ```

### Project will be available at localhost:8000
